# OI 对拍程序

## PairClap

一个简易的对拍工具，希望能有点用。

## 系统要求
Win8 及以上

## 开始使用

首先克隆此库，在 `BF.cpp` 中写上暴力代码，`Std.cpp` 中写上你的正解，`Data.cpp` 中写上相应的数据生成。**无需写 `freopen`**，然后编译 `Clap.cpp`，得到 `exe` 文件，在 `cmd` 中运行 `Clap.exe` 即可。

本库附带 `Random.h`，具有一些基本的生成功能。另外，`Clap.cpp` 中可以更改测试次数与进程数，可以加快对拍速度。**需要 g++ 编译环境！**
